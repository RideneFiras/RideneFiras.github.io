Edits commit straight to RideneFiras.github.io and go live in about a minute. Token never leaves this browser.

  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Repository access: only RideneFiras.github.io
  3. Permissions → Repository → Contents: Read and write, everything else: none
  4. Paste it here. It is stored in this browser's localStorage only.

Tip: content is plain JSON, the same files as data/*.json. Invalid JSON is refused before it ever reaches the repo.
