Edits commit straight to RideneFiras.github.io and go live in about a minute. The token is encrypted with your PIN and never leaves this browser.

  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Expiration: 90 days is a good default (you can mint a new one any time)
  3. Repository access: only RideneFiras.github.io
  4. Permissions → Repository → Contents: Read and write, everything else: none
  5. Paste it here with a PIN. Both stay on this device; the token is stored AES-GCM encrypted.

Tip: content is plain JSON, the same files as data/*.json. Invalid JSON is refused before it ever reaches the repo. Lock drops the decrypted token from memory.
