# firasridene.tech

My personal portfolio — AI engineer, generative AI & multi-agent systems.

Built with [Astryx](https://github.com/facebook/astryx), Meta's open-source, agent-ready design system, on top of React, TypeScript, and Vite. Content is fed from JSON files in `src/data/`.

## Development

```bash
npm install
npm run dev      # dev server
npm run build    # production build to dist/
```

Live at [firasridene.tech](https://firasridene.tech).
